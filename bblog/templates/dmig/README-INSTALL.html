Installation

Dominic Frohlof | May 21, 2005

Installing this template to your bBlog is very easy:

  1. Upload the "dmig" folder by FTP to /bblog/templates/
  2. In your admin section, go to options, under templates select "dmig".
  3. Save options. You're done. :)

This is a first beta release yet. If you experience problems with this template, please check my homepage or the bBlog forum for a recent release or update.

If the problem still exists, I am happy to hear from you with detailed information, and if possible a screenshot.

Compatibility

Dominic Frohlof | May 21, 2005

This template was tested on bBlog version 0.7.4 and 0.75.

It has support for RTL, so for languages such as Arabic or Hebrew.

It should be compatible with future versions, but it might be not.
